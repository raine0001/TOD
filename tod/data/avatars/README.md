# Avatar Store

Place your portrait photo here as `user-avatar.jpg` before running the spokesperson pipeline.

## Requirements
- JPEG or PNG, minimum 512×512 pixels
- Face clearly visible, well-lit, front-facing preferred
- File name: `user-avatar.jpg`

The jungle-spider demo preset references this file at:
`tod/data/avatars/user-avatar.jpg`

Run `Setup-TODSpokesperson.ps1` after placing your photo here.
